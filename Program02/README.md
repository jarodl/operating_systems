CS314: Operating Systems
========================

## Project02 ##

### Without semaphores ###

Compile the example that does not use semaphores by doing:

`cd before_sem`

and then:

`make && ./racecond`

### Using semaphores ###

Compile the example that does use semaphores by doing:

`cd after_sem`

and then:

`make && ./semaphore`
